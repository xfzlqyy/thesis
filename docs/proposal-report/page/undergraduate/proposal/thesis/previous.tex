    \par {\zihao{4}\bfseries 二、指导教师对文献综述、开题报告、外文翻译的具体要求：}
    \vspace{0.8em}

    {\zihao{5}
    \par 1. 文献综述和开题报告的一般要求：
    \par （1）查阅国内外相关文献资料 10 篇以上，其中外文文献不少于 5 篇，近 5 年文献不少于 3 篇。在此基础上完成文献综述，正文不少于 4000 字。
    \par （2）完成与3D Gaussian Splatting（3DGS）或相关神经渲染方法有关的外文翻译，正文（中文）不少于 3500 字。
    \par （3）完成开题报告，正文不少于 4500 字，内容应做到问题明确、方案可行、技术路线清晰、进度安排合理。

    \vspace{0.6em}
    \par 2. 对文献综述的具体要求：
    \par （1）了解鱼眼相机的成像模型、畸变特点、标定方法及其在室内场景采集中的优势与局限，理解高保真室内场景重建在几何一致性、纹理保真、视角覆盖、遮挡处理和光照变化等方面的关键问题。
    \par （2）梳理室内场景重建相关方法的发展脉络，重点总结传统多视图几何方法、NeRF 类方法、3DGS 类方法以及面向鱼眼或广角输入的适配研究，阐述其国内外研究现状。
    \par （3）比较已有方法在标定精度、重建质量、渲染效率、训练代价、细节恢复能力及工程可实现性等方面的差异，说明基于鱼眼相机和 3DGS 进行高保真室内场景重建的研究意义。
    \par （4）为后续毕业论文工作做好知识储备，熟悉相机标定、多视图几何、三维场景表示、神经渲染与实验评测的基本理论和常用工具。

    \vspace{0.6em}
    \par 3. 对开题报告的具体要求：
    \par （1）分析当前用于室内场景重建的主要网络架构和表示方式的种类与特点，重点说明 NeRF、3DGS 及其面向鱼眼输入时的适配思路。
    \par （2）给出基于鱼眼相机的高保真室内场景重建设计方案，明确视频采集、相机标定、关键帧或图像序列生成、数据预处理、3DGS 重建、渲染与评估等核心环节，并选择合理的处理算法和网络架构。
    \par （3）对论文拟完成的工作形成清晰认识，给出具体研究方案，预见可能遇到的问题和困难，如鱼眼畸变建模、视角覆盖不足、纹理细节缺失、动态物体干扰及重建质量评价等，并提出相应应对思路。
    \par （4）提出论文的预期目标，包括实现可复现的鱼眼室内场景重建流程，并在重建保真度、渲染效果或效率方面取得具有分析价值的实验结果。
    \par （5）安排好论文进度，对文献调研、方案设计、系统实现、实验分析、论文撰写与修改等阶段给出详细规划。
    \par （6）如实说明本人已开展的调研、标定、数据采集、预处理或初步重建实验工作，并总结已取得的阶段性结果。}
